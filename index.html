PHP/DB - Schnellkurs

test.php

Die Seite führt eine Reihe von PHP-Operationen aus und demonstriert grundlegende PHP-Konzepte wie den Umgang mit Variablen, Bedingungen, Schleifen und Funktionen. Im HTML-Teil des Dokuments wird das Ergebnis dieser Operationen angezeigt.

Zusammengefasst demonstriert der Code grundlegende PHP-Funktionen und zeigt, wie man PHP innerhalb eines HTML-Dokuments für dynamische Webseitengestaltung verwenden kann.

crud_direkt.php

Die Seite führt eine Reihe von CRUD-Operationen (Create, Read, Update, Delete) auf einer Datenbanktabelle aus. Die Operationen umfassen das Abrufen aller Datensätze, das Einfügen eines neuen Datensatzes, das Lesen eines Datensatzes mit einer bestimmten ID, das Lesen aller Datensätze, die einen bestimmten String enthalten, und das Löschen eines Datensatzes mit einer bestimmten E-Mail-Adresse.

crud_api.html

Die HTML-Seite, die die CRUD-Operationen über die API-Schnittstelle aufruft und die Ergebnisse anzeigt. Die eigentliche Funktionalität liegt im Zusammenspiel der Dateien crud_api.html (Eingabe der Werte), script.js (Übertragung der Werte) und api.php (Verarbeitung der Werte). Der Einfachheit halber werden die zurückgegebenen Werte nicht angezeigt. Sie können aber via JavaScript in der Konsole ausgegeben werden.

script.js

Die JavaScript-Datei, die die Interaktivität und Funktionalität der Webseiten im Projekt implementiert.

api.php

Die Seite stellt eine API-Schnittstelle für CRUD-Operationen auf einer Datenbanktabelle bereit. Die Operationen umfassen das Abrufen aller Datensätze, das Einfügen eines neuen Datensatzes, das Lesen eines Datensatzes mit einer bestimmten ID, das Lesen aller Datensätze, die einen bestimmten String enthalten, und das Löschen eines Datensatzes mit einer bestimmten E-Mail-Adresse. Die API-Schnittstelle ermöglicht es anderen Anwendungen, auf die Datenbank zuzugreifen und CRUD-Operationen durchzuführen.

index.php

Die Startseite des Webprojekts, die eine Übersicht über die verfügbaren Seiten und deren Funktionen bietet.
